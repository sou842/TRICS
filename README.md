# Forten Construction

A modern Next.js project for Forten Construction, featuring a high-performance landing page with Tailwind CSS 4 and Framer Motion (conceptually, though currently using standard CSS animations and Lucide icons).

## Tech Stack

- **Framework:** Next.js 15+ (App Router)
- **Styling:** Tailwind CSS 4
- **Icons:** Lucide React
- **Components:** Radix UI (Shadcn UI base)
- **Fonts:** Inter & Inter Tight (via Google Fonts)

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `src/app`: Next.js App Router (Layouts, Pages, Styles)
- `src/components`: React components (UI and Site-specific)
- `src/assets`: Static assets like images
- `src/hooks`: Custom React hooks
- `src/lib`: Utility functions and configuration
